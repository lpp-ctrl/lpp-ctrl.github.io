centos7安装snort

 
1.下载daq包和snort包
wget https://www.snort.org/downloads/snort/daq-2.0.6.tar.gz
wget https://www.snort.org/downloads/snort/snort-2.9.14.1.tar.gz
 
 
2.解压包
tar xvzf daq-2.0.6.tar.gz
cd daq-2.0.6
./configure && make && sudo make install
 
tar xvzf snort-2.9.14.1.tar.gz
cd snort-2.9.14.1
./configure –enable-sourcefire && make && sudo make install（报错）
 
 
3.安装snort的时候报错
报错信息，如下图所示

使用
./configure –disable-open-appid
make && make install即可解决此错误
 
wget https://www.snort.org/downloads/community/community-rules.tar.gz -O community-rules.tar.gz
tar -xvzf community-rules.tar.gz
mv community-rules/ /data/server/snort/rules
 
 
使用snort -V查看一下版本
